## 소개
단색 LED는 재미있고 교육적인 시연에 적합합니다. 또한 학생들의 소규모 탐구 활동에도 매우 좋습니다. 어두워진 교실에서의 밝은 색상은 학생들이 확실히 기억할 만한 분위기를 조성합니다.

## 장비
모든 실험에는 빨간색, 녹색, 파란색의 단색 LED가 필요합니다. 실험별로 다음 구성 요소가 필요합니다.

| 실험 | 장비 |
|---|---|
| 스펙트럼 만들기 | 분광계, 측정 소프트웨어 |
| 빛의 중첩 | 흰색 스크린 |
| 회절 스펙트럼 보여주기 | LED TV 또는 노트북 화면 |
| 빛의 산란 | 저렴한 흰색 플라스틱 축구공 |
| 램프 렌즈의 굴절력 계산 | 렌즈 홀더 |

```{tip}
- 사용된 LED는 다양한 온라인 쇼핑몰에서 구입했습니다. LED 램프가 생성하는 빛의 강도를 확인하고 빛을 많이 내뿜는 것을 선택하세요.
- 저렴한 축구공 대신 탁구공을 사용할 수 있습니다.
```

## 준비
수행할 시연을 결정하고 다음 재료들을 준비합니다.
1. 분광계가 연결된 상태에서 소프트웨어를 실행합니다.
2. 교실을 충분히 어둡게 할 수 있는지 테스트합니다. 빛을 중첩해 보고 명확하게 보이는지 확인합니다. 필요한 경우 배터리를 교체합니다.
3. LED 램프를 쌓아 올리고 동시에 또는 개별적으로 TV나 노트북 화면을 겨냥하여 회절 스펙트럼이 보이는지 확인합니다.
4. 공 아래에 LED 램프를 놓고 공이 굴러가지 않도록 고정합니다.
5. LED 램프를 분해하여 렌즈를 제거합니다.

## 물리 배경
1. 램프의 스펙트럼은 상당히 단색에 가깝습니다.
2. 빛의 혼합 색상에 관한 물리학은 잘 알려져 있습니다.
3. 파장이 짧을수록 편향이 작습니다. {numref}`그림 {number} <demo89_fig4>`에서 LED TV의 회절 스펙트럼이 이중 슬릿과 단일 슬릿의 조합임을 알 수 있습니다.
4. LED 램프의 빛은 흰색 공 내부로 들어가 내부에서 확산 산란됩니다. 그러면 공 전체에서 램프의 색상이 방출됩니다.
5. {numref}`그림 {number} <demo89_fig6>`은 배율이 어떻게 결정되는지 보여줍니다. 다른 상황에서 우리는 $N = -3.0$임을 발견했습니다. 렌즈에서 종이까지의 거리: 측정된 $v = 0.68$ cm. $b$의 경우 $-2.0$ cm를 얻습니다 ($b$는 허상). 렌즈 공식을 사용하면 $f = 1.0$ cm 또는 $S = 1.0·10^2$ dpt입니다.
